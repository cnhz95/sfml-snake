Detailed Description

Class for the game engine responsible for the movement of the snake, checking for collisions, spawning of the fruit, updating the score, etc.

Constructor & Destructor Documentation

◆ Game()

Game::Game ( )

Default constructor. Creates the game object and puts it in the main menu state. Instantiates the world, snake, fruit and score managed objects. Sets the snake's start direction to the right and the time step to default. Initializes the iterators pointing to it's head and it's tail.
